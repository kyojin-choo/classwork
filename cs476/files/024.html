How to Obtain SIR-C Software

If you wish to obtain an 8mm tape with the SIR-C CEOS_READER software , the data decompression software and the SIR-C/AIRSAR conversion software , these are the steps that have to be taken:

  1. print out the software licenses

  2. fill out and sign

  3. fax all portions to :

    4. Attention: Ellen O'Leary
    Radar Data Center
    Jet Propulsion Laboratory
    1-818-393-2640

  4. An 8mm tape containing the software (with manuals) will be mailed to you.
If you have already signed and returned the license agreement, and want to get the latest version by ftp, then contact Ellen O'leary for further instructions :

email : ellen.oleary@jpl.nasa.gov
